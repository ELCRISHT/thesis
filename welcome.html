About IntelliGrade

IntelliGrade is an innovative platform designed to forecast academic performance by analyzing students' dependency on AI tools. Utilizing machine learning algorithms and psychological theories, it provides educators and administrators with insights to support student success in an AI-integrated learning environment.

Built using Cognitive Load Theory (CLT), Self-Determination Theory (SDT), and Social Cognitive Theory (SCT), IntelliGrade generates personalized recommendations based on usage behavior and cognitive patterns. By identifying risk indicators and learning impacts related to AI reliance, the system helps mitigate potential academic challenges while promoting effective AI integration in education.
